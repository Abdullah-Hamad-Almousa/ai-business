# Abdullah Almousa — Machine Learning Studio

The official business website for **Abdullah Almousa**, an ML specialist based in Riyadh, Saudi Arabia.

Live site: [ai.modelai.website](https://ai.modelai.website)

---

## About

This website showcases professional machine learning and AI services offered by Abdullah Almousa. It is a single-page application built with React, Vite, and Tailwind CSS, featuring a glassmorphism design on a light Apple-inspired background with animated neural network visuals.

## Services Showcased

- Custom Machine Learning model development
- Computer Vision solutions
- Natural Language Processing (NLP)
- Data analysis and visualization
- AI consulting and strategy
- Model deployment and MLOps

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion (scroll animations)
- Custom neural network canvas background
